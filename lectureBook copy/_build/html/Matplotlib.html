1. Matplotlib Basics#

1.1. Why Graph Your Data?#

Within your data frame, it is sometimes hard to spot a correlation between 2 different attributes. Using Matplotlib to graph different attributes can help you analyze and interpret your data better.

1.2. Some Quick Options#

These are many graphs that are very easy to use through Matplotlib. You will sometimes have to test out a few before you find what you like.

1.2.1. Histograms#

Histograms work best when displaying continuous and fully numerical data, they are most commonly used to show frequency distributions in a neat and concise manner.

import matplotlib.pyplot as plt
import numpy as np
import pandas as pd

x = np.random.normal(170, 10, 250)
df = pd.DataFrame(x)

print(df)

plt.hist(df)
plt.title("Random df")
plt.ylabel("frequency")
plt.xlabel("number")
plt.show()

1.2.2. Bar graphs#

Bar graphs are good when performing a comparison of metric values across different groups of your data. It is also good to use if you wanna create designated subgroups for your primary data groups.

import matplotlib.pyplot as plt
import numpy as np

df = pd.DataFrame({"Fruits": ["Apples", 'Bananas'], 
                   "Values": [12, 17]
                  },index = [1,2])
plt.bar(df["Fruits"],df["Values"])
plt.title("Random df")
plt.ylabel("Amount")
plt.xlabel("Fruits")
plt.show()

1.2.3. Pie charts#

Pie charts are very common. They are visually and contextually simple. They are ideal if the sum of the individual parts adds up to a meaningful whole, and is built for visualizing how each part contributes to that whole.

import matplotlib.pyplot as plt
import numpy as np

data = np.array([35,25,20,20])
myLabels = ["Apples", "Banana", "Cherries", "Peaches"]
plt.pie(data, labels = myLabels)
plt.show()

1.2.4. Scatter plots#

A Scatterplot uses dots to represent values for two different numeric variables and is a great way to visually identify a correlation between them.

import matplotlib.pyplot as plt
import pandas as df
import numpy as np

x = df.DataFrame([5,7,8,7,2,17,2,9,4,11,12,9,6])
y = df.DataFrame([99,86,87,88,111,86,103,87,94,78,77,85,86])

plt.scatter(x, y)
plt.title("x vs y")
plt.ylabel("y")
plt.xlabel("x")
plt.show()

1.3. Tips and Ticks With Paramaters#

1.3.1. Colour#

You can easily add color to your graphs when using Matplotlib. Colors can be identified by the library as abbreviations, names, hex values, and RGB/RGBA values.

For our usage using abbreviations will be the most beneficial, some useful ones to keep in mind are:

Abbrevation

colour

b

blue

g

green

r

red

c

cyan

m

magenta

y

yellow

k

black

w

white

To apply the colors to the graph all you need to do is address the color parameter with the options shown above.

import matplotlib.pyplot as plt

x = [5,10,15,20]
y = [10,20,30,40]

plt.plot(x,y, color='r') # red
plt.show()

1.3.2. Scale#

There are many ways to change your scale, and it may be necessary depending on the size and range of your data. To do this you can:

  • re-write the ticks

import matplotlib.pyplot as plt
# here we chose our ticks as 1-5 for both the x and y axis
plt.xticks(range(1, 5)) 
plt.yticks(range(1, 5))

import matplotlib.pyplot as plt
# Setting the x-axis to 0-5
# and y-axis to 1-15
plt.axis([0, 5, 1, 15])
